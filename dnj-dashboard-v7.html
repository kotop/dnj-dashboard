Ключевые рекомендации по улучшению методики
⚡
Single Source of Failure (П1, П2)
Полная зависимость от ЕИС Добро.рф. Рекомендация: добавить социологический компонент ВЦИОМ как страховку (вес 20%).
⚡
«Мёртвые» НКО в П3
Знаменатель — весь реестр МЭР, включая фактически неактивные НКО. Рекомендация: фильтровать по ИНН ФНС — только ЮЛ с активным статусом.
⚡
Нет retention в П1
Методика фиксирует новых участников, но не удержание. Рекомендация: поднять Т-3.11 (retention 12 мес.) как обязательный компонент П1.
⚡
П6 не измеряет загрузку ДЦ
Фиксируется наличие, но не востребованность. Рекомендация: добавить индикатор «обращений на 1000 жителей».
⚡
Нет дифференциации целей по типу регионов
14% охват для Москвы и Чукотки — некорректно. Рекомендация: кластеризация регионов (мегаполисы / средние / малые / северные).
⚡
Веса сводного индекса
Веса установлены только экспертно. Рекомендация: провести факторный анализ на исторических данных 2022–2025 для валидации весов.
